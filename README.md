# Catalog Menu New Design

Локальная статическая версия главной страницы CRM с восстановленным интерфейсом Kanban, каталог-меню и поддержкой светлой/тёмной темы.

## Что внутри

- восстановленный экран `Главная страница`
- рабочее левое каталог-меню (раскрытие/сворачивание)
- режимы `expanded / minified` для сайдбара
- всплывающее подменю в свернутом режиме
- логика `Показать все / Скрыть неактивные`
- закрепление пунктов меню
- Drag & Drop для типов документов в Kanban navbar
- стили для light/dark версии
- локальные иконки Tabler + Font Awesome

## Быстрый запуск

```bash
cd "/Users/memodlike/Documents/Catalog menu new design"
python3 -m http.server 4173
```

Открыть в браузере:

`http://127.0.0.1:4173/Главная%20страница.html`

## Основные файлы

- `Главная страница.html` — основной экран и inline-стили Kanban
- `Главная страница_files/myCss.css` — кастомные стили проекта
- `Главная страница_files/GlobalScripts.js` — меню, fallback-логика, поведение UI
- `Главная страница_files/Kanban.js` — Kanban, фильтры, Drag & Drop, счетчики

## Примечание

Проект предназначен для локального статического запуска. Backend-зависимые API частично переведены на fallback-режим для корректной демонстрации интерфейса.
