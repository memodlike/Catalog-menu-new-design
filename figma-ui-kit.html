Поведение компонентов

Каталог меню: стрелка вправо = свернуто; при раскрытии поворот вниз; клик повторно сворачивает.
Свернутое меню: только иконки, одинаковый размер и одинаковые вертикальные интервалы (без прыжков).
Личный кабинет / Мои документы: по умолчанию показываются активные и pinned-пункты; «Показать все» раскрывает все, но не ломает baseline-логику.
Pin: закрепляет пункт независимо от счетчика; pinned не скрывается.
Drag&Drop (4 типа доков): зона слева в navbar; во время drag активный элемент приподнят, целевой слот подсвечивается индикатором before/after.
Карточки Kanban: overdue всегда в формате «Просрочено: N дн.»; для action-сценариев красный счетчик/статус.
Темы: Light и Dark используют одинаковую геометрию, только палитра и контраст адаптированы.
Рекомендация для разработчика: сохранять DnD-порядок и pinned-state в localStorage по ключам модуля; при ререндере сначала применять порядок, затем фильтрацию активных/неактивных пунктов.
